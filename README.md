## ğŸ§  What This Project Does

This project provides a secure REST API that accepts a Base64 encoded MP3 audio and determines whether the voice was spoken by a real human or generated by an AI system.

The detection is performed using **speech signal processing heuristics** such as:

* Pitch stability
* Spectral smoothness
* Energy consistency
* Temporal variation

AI generated voices typically show **unnaturally stable acoustic patterns**, while human voices exhibit natural irregularities.

## ğŸ§ª How Detection Works (High Level)

The system analyzes acoustic features rather than words, making it language independent.

Human speech:

* Natural pitch variation
* Micro pauses and breathing noise
* Irregular spectral distribution

AI speech:

* Overly consistent pitch
* Extremely clean waveform
* Stable frequency distribution

This makes the model effective across multiple languages without translation.
